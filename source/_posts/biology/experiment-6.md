| 实验目的 | 学会用燃烧法测定食物中的能量 | | |
| 实验器材 | 花生种子、水、实验架、铁夹、锥形瓶（50Ml）、石棉网、托盘天平、酒精灯、火柴、挑针、温度计、湿毛巾 | | |
